# Effective Mobile DevOps Test

## Описание

Простое веб-приложение, развернутое с использованием Docker и Docker Compose.

Состоит из:
- backend (Python HTTP сервер)
- nginx (reverse proxy)

nginx принимает HTTP-запросы на порт 80 и проксирует их в backend.
